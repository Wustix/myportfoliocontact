Group Project 3 - FAST Towing App

FAST Towing


Bootstrap | Javascript | Node.js | Express | Passport.js | React.js | Nodemailer | SMS Messaging | MongoDB | Google Maps Geo-Location


FAST Towing is a Node.js/React client app that allows Users to request roadside assistance. Google Geo-Location provides accurate GPS location, while SMS Messaging/Nodemailer allow for real-time contact with roadside service providers. Create an account utilizing Passport.js login with MongoDB to be ready for any situation at a moments notice.
